# AWS Networking Day - Advanced

AWS Networking Day - Advanced 主要是针对已经对 AWS Networking 非常熟悉且需要构造复杂网络解决方案的用户。
本次训练营分为两部分：

- 基于 Direct Connect 在 On-Premise 数据中心以及 AWS 云上构建复杂的 Hybrid 互联网络。

- 基于 Transit Gateway 在 AWS 云上构建复杂的 VPC 网络。
